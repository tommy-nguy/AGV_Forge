# AGV Forge

Offline-First End-to-End Video Automation Pipeline

## Overview
AGV Forge is a local-first video automation factory that transforms raw footage or text content into publish-ready videos with AI-powered planning, voice synthesis, image generation, and smart editing.

## Features
- **System 1**: Re-edit raw videos with AI-driven timeline intelligence.
- **System 2**: Create cinematic or knowledge videos from text using Google Flow and character voice.
